====================================================
  SHIVAM FOOTWEAR SHOP - Desktop Management System
  Offline app - No internet required for daily use
====================================================

FIRST TIME SETUP (one time only):
---------------------------------
1. Unzip/copy the full ShivamFootwear folder to:
   D:\SHIVAM_FOOTWEAR\ShivamFootwear\
2. No Java install needed - runtime is bundled with the app.
3. Double-click "ShivamFootwear.exe" (recommended)
   OR double-click "Start Shivam Footwear.bat"
4. Login with PIN: 1234
5. Go to Settings and update shop name, address, phone, PIN

DAILY USE:
----------
Morning:  Double-click "ShivamFootwear.exe"
          Login with your PIN
          Use Counter Sale / POS to bill customers
          Use Stock Management to add new shoes
          Use Sales Register to reprint old bills
Evening:  Close the app window
          OR run "Stop Shivam Footwear.bat"

IMPORTANT RULES:
----------------
- Do NOT delete the "data" folder - all shop data is stored there
- Backup weekly: Settings -> Backup Now
- Do NOT run the app twice at the same time
- Internet is NOT required for daily use
- The app opens in a desktop window (not Chrome/browser)

TROUBLESHOOTING:
----------------
- Blank window: Wait 30 seconds, then close and start again
- Port error: Run Stop bat first, then start again
- Forgot PIN: Contact your software provider
- If EXE missing: Use Start Shivam Footwear.bat
